## What is MyTemplate?

MyTemplate is a comprehensive template for creating professional documentation sites for Julia packages. It demonstrates best practices for:

- Organizing documentation
- Creating interactive examples with Pluto
- Serving web content with Genie
- Auto-generating API docs with Documenter.jl

## Features

✨ **Interactive Notebooks**: Run Pluto notebooks directly in your browser  
📚 **Auto-Generated Docs**: API documentation generated automatically  
🚀 **Fast Web Server**: Built on Genie Framework  
🎨 **Responsive Design**: Works on all devices  
📱 **Mobile-Friendly**: Optimized for mobile viewing  

## Getting Started

The fastest way to get started:

```julia
using MyTemplate
MyTemplate.serve()
```

Then open your browser to `http://localhost:8000`.
